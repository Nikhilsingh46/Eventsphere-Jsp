# EventSphere 🎉

A Collaborative Event Management Platform built using **JSP, Java, MySQL, HTML, CSS, and JavaScript**.

## Features
- User Dashboard
- Admin Dashboard
- Event Management
- College-specific events
- Dynamic JSP pages

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: JSP, Servlet, JDBC
- Database: MySQL
- Server: Apache Tomcat

## How to Run
1. Import project in Eclipse
2. Configure Apache Tomcat
3. Setup MySQL Database
4. Run on Server
